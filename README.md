# Weather App (PERN Stack)

A simple **Weather App** built with the **PERN stack** (PostgreSQL, Express, React, Node.js).  
You can search for cities, view current weather, and see search history.

## Features

- Search weather by city
- Show temperature, humidity, wind, and weather description
- Search history (last 10 searches)
- Backend + Frontend fully connected

## Technologies

- **Frontend:** React, Axios
- **Backend:** Node.js, Express
- **Database:** PostgreSQL
- **Authentication:** (optional, if implemented)
